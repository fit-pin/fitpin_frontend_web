## 사용법

1. [**node.js & npm설치**](https://nodejs.org/en)

2. **프로젝트 폴더에서 npm 종속성 설치**

    ```bash
    npm i
    ```

3. **npm 으로 시작**

    ```bash
    npm run start
    ```
